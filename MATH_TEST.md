# Math Formula Test

## Display Math (Block)

The following is a block math formula:

$$\frac{\sum_{i=1}^{n} x_i}{n} = \text{ממוצע}$$

## Inline Math

In Hebrew: פיתוח מחמתי הכולל טורים כמו $\sum_{i=1}^{n} x_i$ וגם שברים כמו $\frac{a}{b}$.

## More Examples

- Mean: $\bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i$
- Pythagorean: $a^2 + b^2 = c^2$
- Square root: $\sqrt{x^2} = |x|$

## Complex Formula

$$E = mc^2$$

$$\text{Distance} = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$$
