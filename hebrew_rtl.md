## הקדמה

זה קובץ markdown שנכתב בעברית עם תמיכה מלאה בכיווניות מימין לשמאל. 
המטרה של קובץ זה היא להדגים כיצד לכתוב תיעוד בעברית עם יישור תקין לימין.

### מה זה RTL?

RTL משמעותו Right-To-Left (מימין לשמאל). זו הכיווניות הטבעית של שפות שונות כמו:
- עברית
- ערבית
- פרסית
- אורדו

## דוגמאות

### דוגמה 1: טקסט בסיסי

זה טקסט רגיל בעברית. אנחנו יכולים להשתמש ב**הדגשה חזקה** וב*הדגשה קלה*.
כמו כן, אנחנו יכולים להשתמש ב~~טקסט שכיב~~ ובקישורים.

### דוגמה 2: רשימה מסודרת

תהליך העבודה:

1. ההכנה - הכנת כל החומרים הדרושים
2. התכנון - תכנון מפורט של העבודה
3. ההוצאה לפועל - ביצוע המטלות
4. בדיקה - בדיקת העבודה שהתבצעה
5. הגמר - סיום וסיכום

### דוגמה 3: רשימה לא מסודרת

דברים חשובים:

- רשימה ראשונה
  - תת-רשימה א
  - תת-רשימה ב
- רשימה שנייה
- רשימה שלישית
  - תת-רשימה ג
  - תת-רשימה ד

## קוד

### דוגמה: Python

```python
# קוד עברי בהערות
def שלום(שם):
    """
    פונקציה שמדפיסה שלום
    """
    print(f"שלום {שם}!")

# קריאה לפונקציה
שלום("עולם")
```

### דוגמה: JavaScript

```javascript
// משתנים בעברית בהערות
const ברכה = "שלום עולם";
const מספר = 42;

console.log(ברכה);
console.log(`המספר הוא: ${מספר}`);
```
